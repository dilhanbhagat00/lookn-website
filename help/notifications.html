Notifications & reminders

Last updated May 19, 2026

Lookn sends notifications for task reminders, streak nudges, friend activity, and the occasional update. Everything is opt-in by category and you can mute it all in one tap.

Turn on notifications

First time you open Lookn, we ask permission via your phone's system prompt. Tap Allow to get reminders.

If you said no the first time and want to turn them on later:

  • iOS: Settings (system) → Notifications → Lookn → toggle Allow Notifications.
  • Android: Settings (system) → Apps → Lookn → Notifications → toggle on.

Then open Lookn and reload to register your push token.

Tune which notifications you get

Inside the app: Settings → Notifications. Toggle individual categories:

  • Task reminders — daily nudges for tasks you haven't completed yet.
  • Streak nudges — late-day "don't break your streak" pings.
  • Friend activity — feels received, friend requests, shared task accepts.
  • Quest reminders — daily quest unlock.
  • Adventure updates — adventure milestone notifications.

Turn everything off

Two ways:

  • Settings → Notifications → toggle each category off.
  • System Settings → Notifications → Lookn → off. Hard mute.

Notification times

Task reminders fire at the time of day each task is scheduled for. The "late-day streak nudge" defaults to about 9pm local time. You can change that in Settings → Notifications → Quiet hours / reminder time.

Tip: If you're getting too many friend pings, mute the Friend activity category specifically — it's the loudest one for users with lots of friends.

FAQ

I'm not getting any notifications.

Check three things: (1) system Settings → Notifications → Lookn is allowed, (2) Do Not Disturb / Focus mode isn't blocking them, (3) in-app Settings → Notifications has the category enabled. Re-allow if you ever revoked permission.

I'm getting reminders for friends who left.

Force-close and reopen the app — the Friends list refreshes from the server and clears deleted-friend cards. Reminders stop on the next sync.

Can I get a Sunday-off mode?

Not as a built-in toggle yet — but Quiet hours can give you a quiet window every day.
